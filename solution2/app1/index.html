Web Component Demo

This React app is mounted as a web component. You can use it in any HTML page with the following tag:

<react-app></react-app>
Note: To use this component in other applications, you need to:
  1. Include the built JavaScript bundle from the dist folder using:
    <script type="module" src="path/to/app1/dist/assets/index.js"></script>
  2. Add import maps for React dependencies (see head section of this HTML)
  3. Use the <react-app> tag in your HTML

The bundle has a static filename (without hash) for consistent importing and excludes React libraries to reduce bundle size and allow sharing.

Using as a Standard React App

This application can also be used as a standard React app by uncommenting the React mounting code in main.jsx and adding a root div to your HTML:

<div id="root"></div>

See the README.md and main.jsx comments for more details.

Import Maps & External Dependencies

This web component uses import maps to load React and ReactDOM from a CDN rather than bundling them:

<script type="importmap"> { "imports": { "react": "https://esm.sh/react@18.2.0", "react-dom": "https://esm.sh/react-dom@18.2.0", "react-dom/client": "https://esm.sh/react-dom@18.2.0/client", "react/jsx-runtime": "https://esm.sh/react@18.2.0/jsx-runtime" } } </script>

This provides several benefits:

  • Smaller bundle size as React is not included
  • Multiple micro-frontends can share the same React instance
  • Consistent React version across components
  • Better caching as dependencies are loaded from CDN
